# runner-unity
# Игра раннер созданная на C# и Unity
## Технологии:
### C#
### SQLITE
### Unity
## Управление:
### Прыжок - пробел
### Бустер - E
### Скольжение - S
## Рейтинг:
### В игре присутствует рейтинговая система, во вкладке "Рекорды" можно увидеть рейтинг игроков локалькой сессии
